# Debug Life

## 为什么要写这本书？
目前我们所处的社会环境正处于时代巨变之中，一方面中国的社会经济发展到达了一定的阶段，另一方面，科技的发展正在逐渐带来生产力的巨大飞跃。尤其是人工智能，已经重塑这个时代。这也令本就很多迷茫的年轻人，面临着前所未有的困境。在互联网上、在生活中，经常能收到很多同学、朋友、网友甚至是陌生人关于“人生”话题的分享或咨询。他们或是初入职场，或是还在求学阶段，或是正在思考谈婚论嫁的人生大事。他们或是意气风发，或是意志消沉，或是焦虑不安。以我作为一名沉浸coding多年的程序员的视角来看，解决生活问题与解决coding问题的本质是一样的，一言以蔽之，就是“拆解问题，寻找最优解”。说起来简单，但是说实话，真正能看清问题是什么的人少之又少。coding领域如此，生活亦是如此。大部分人对问题的思考浮于表面，缺乏架构师思维，不能深入结构性问题上。这样就找不到解决问题正确的“道”。从“术”的角度来看，寻找问题的最优解，或者说就是学会“投机取巧”，不能“教条主义”，而“投机取巧”是需要经过严谨的论证和分析的，对思维逻辑的严密性有着很高的要求。一名优秀的程序员，所提交的每一行代码，都要清楚地知道为什么要这样写，有没有其他方法，为什么采用这种方法。人生中的每一次选择和行为，也应当如此。人生的脉络也应该清晰、整洁、合理，在我们审视生活的时候，就应该像在“Debug”程序一样，每一个行动都应该去想一想最优解是什么。思考的复杂度提高，带来的是未来人生的更易维护。而作为一名优秀的程序员，开源自己的知识和想法，亦是我的人生理想。因此特写此书，来阐述我对生活和当前中国社会的理解，来帮助与我处于相似人生阶段的普通人提供一些参考。

## 本书的逻辑线

### Catch Error
认清当下影响我们生活“爽感”的真正的困境究竟是什么。

### Code Review
核心部分，审视人生的几个重要课题：活着、工作、学习、金钱、情感、家庭、婚姻、社会价值等等。

### Refactor
探讨当AGI时代到来时我们的生活方式和生活态度会如何变化。
