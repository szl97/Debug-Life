# 4. 欲望是人生的目标函数

## 4.1 活着：一个你没有选择的进程

你为什么会存在于这个世界上？

不是哲学意义上的"存在的意义"，而是一个更基础的事实：**你从未选择过要出生。**

某一天，你的人生进程就被系统启动了，没有人问过你的意见。你睁开眼睛，发现自己已经在运行了。

这个进程有几个特点：

**第一，启动是被动的。** 你没有选择出生在哪个国家、哪个家庭、哪个时代。这些初始参数在你有意识之前就已经被设定好了。

**第二，运行是强制的。** 一旦启动，你就必须持续运行，直到进程终止。中途强制退出（自杀）在大多数文化和法律中都是被禁止或不被鼓励的。

**第三，终止是必然的。** 不管你愿不愿意，这个进程最终会结束。时间是唯一的硬性约束。

所以，**活着不是一个结果，而是一个状态。** 它是一段runtime，不是一个return value。

既然我们无法选择是否启动，也无法阻止最终的终止，那我们唯一能做的，就是**优化这段运行时的体验**。

问题来了：怎么定义"体验好"？用什么指标来衡量？

## 4.2 目标函数：活着就是为了"爽"

在机器学习里，有一个核心概念叫**目标函数**（Objective Function）。它定义了模型要优化的方向——你想让什么最大化，或者让什么最小化。

目标函数选错了，模型就会往错误的方向狂奔。

人生也需要一个目标函数。**你到底在优化什么？**

有人说是财富，有人说是地位，有人说是知识，有人说是爱情，有人说是自我实现……

但如果我们追问下去：你要财富是为了什么？要地位是为了什么？要爱情是为了什么？

最终的答案都会指向同一个词：**爽。**

财富让你爽，地位让你爽，爱情让你爽，被认可让你爽。你追求的所有东西，本质上都是在追求某种主观上的愉悦感受。

**"爽"是唯一诚实的目标函数。**

这不是享乐主义的宣言，而是一个朴素的事实。你可以给它换一个更高雅的名字——幸福、满足、意义感——但它们的本质都是一种主观体验。

有人可能会反驳：那些为了理想献身的人呢？他们不爽啊。

其实他们也"爽"。为理想献身的人，在实现理想的过程中获得了一种更高层次的满足感。如果完全不爽，没有人能坚持下去。只是他们的爽感来源很多人不理解。

而“为理想献身”之所以在大多数主流文化中都被歌颂和赞扬，是因为他们的行为为主流社会中的更多人创造了“爽”，他们的“爽”建立在了让主流群体“爽”的基础上。

从这个角度来看，并不存在理想多崇高，或者人生多高尚的说法。大家都是在追求”爽“，而人生的”爽“在本质上没有高低贵贱之分。只有社会价值观是否倡导之分。

可能你现在会对此等看法嗤之以鼻，不妨读完本章甚至本书的第二部分后，再回头思考。

**活着的质量，只能用主观感受来衡量。**

客观指标——收入、职位、房产——只是手段，不是目的。如果一个人年入百万但每天痛苦不堪，另一个人月入五千但内心充实愉悦，谁的人生更"成功"？

这里有一个极端的思想实验：

假设有一个人，他有严重的精神疾病，完全活在自己的幻想世界里。在他的世界里，他是国王，拥有一切。他每天都很快乐，从不焦虑，从不痛苦。

从外部看，他是一个"可怜的疯子"。但从他自己的主观体验来看，他的人生是"成功的"——因为他的目标函数一直在输出高值。

这个例子听起来荒谬，但它揭示了一个真相：**如果爽感可以自产自销，那外部世界的一切都不重要了。**

问题是：为什么我们做不到？

为什么我们不能像那个精神病人一样，在自己的脑子里构建一个完美世界，然后永远快乐地活下去？

要回答这个问题，我们需要理解"爽"到底是什么。

## 4.3 爽感的本质：欲望的满足

爽感不是凭空产生的。它是一个**信号**——当某种条件被满足时，大脑释放的奖励信号。

什么条件？**欲望。**

你饿了，吃到食物，爽。
你渴了，喝到水，爽。
你累了，躺下休息，爽。
你孤独，有人陪伴，爽。
你努力工作，获得认可，爽。

**爽感 = 欲望被满足时的反馈。**

没有欲望，就没有爽感。一个对食物没有任何欲望的人，吃山珍海味也不会觉得爽。一个对认可没有任何欲望的人，获得诺贝尔奖也不会觉得爽。

所以，爽感的前提是欲望。**欲望是因，爽感是果。**

这意味着，要理解我们为什么快乐、为什么痛苦，必须先理解我们的欲望是什么、从哪里来。

## 4.4 欲望的起源：从动物性说起

让我们把时间倒回去，回到人类还不是"人类"的时候。

在进化的早期，我们的祖先只是一群普通的灵长类动物。那时候的"欲望"非常简单，完全服务于两个终极目标：**生存和繁衍。**

**生存相关的欲望：**
- 食欲：饿了想吃东西，这驱动你去寻找食物
- 渴欲：渴了想喝水，这驱动你去寻找水源
- 安全欲：遇到危险想逃跑，这驱动你远离威胁
- 休息欲：累了想休息，这驱动你恢复体力

**繁衍相关的欲望：**
- 性欲：想要交配，这驱动你寻找配偶
- 亲子欲：想要保护后代，这驱动你照顾幼崽

这些欲望是**写死在基因里的底层代码**。不需要学习，不需要教育，每个正常的生物体都自带这套程序。

为什么进化会设计出"欲望-满足-爽感"这套机制？

因为它有效。

那些没有食欲的个体，会忘记吃东西，然后饿死。那些没有性欲的个体，不会去寻找配偶，然后绝后。经过千万年的自然选择，只有那些欲望系统正常运作的个体才能存活下来，把基因传递下去。

**欲望是进化给你安装的驱动程序。没有它，你的硬件跑不起来。**

在这个阶段，原始人的快乐很简单：吃饱了、喝足了、安全了、交配了。欲望基本可以"自产自销"——你自己去打猎、自己去找水、自己去找配偶，满足了就爽。

但接下来，事情发生了变化。

## 4.5 欲望的演化：社会性的诞生

大约在几十万年前，人类开始形成**群居社会**。

这不是因为我们喜欢热闹，而是因为**群居能提高生存概率**。一个人打不过狮子，成百上千个人可以。一个人找不到足够的食物，部落可以分工协作。一个人生病了就死了，群体可以相互照顾。

群居带来了生存优势，但也带来了一个副产品：**新的欲望。**

### 4.5.1 被接纳的欲望

在原始部落里，被群体接纳意味着活下去，被群体驱逐意味着死亡。

一个被部落抛弃的原始人，独自面对野兽和饥荒，存活概率极低。所以，那些强烈渴望被群体接纳的个体，会更努力地融入群体，更有可能存活下来。

进化于是筛选出了一种新的欲望：**被接纳的欲望**。

你天生害怕被孤立，天生渴望属于某个群体，天生在意别人是否接受你。这不是你脆弱，是基因在保护你。

### 4.5.2 地位的欲望

部落里的资源是有限的。谁能分到更多的食物？谁能获得更好的交配机会？

答案是：**地位高的人**。

在原始社会，地位通常由力量、智慧、贡献度决定。地位高的个体获得更多资源，有更高的存活率和繁衍成功率。

所以，进化又筛选出了一种新的欲望：**地位的欲望**。

你天生渴望被尊重，天生想要往上爬，天生会和别人比较。这不是你虚荣，是基因在驱动你争夺资源。

### 4.5.3 被需要的欲望

群居社会需要协作。打猎需要配合，建造需要分工，防御需要团队。

那些能被别人需要的个体，更容易获得群体的保护和资源分配的倾斜。一个对部落有价值的人，部落会保护他；一个对部落没价值的人，在资源紧张时会被优先抛弃。

于是，进化筛选出了第三种社会性欲望：**被需要的欲望**。

你天生想要对别人有用，天生在被需要时感到满足，天生在被忽视时感到失落。这不是你在讨好别人，是基因在确保你对群体有价值。

### 4.5.4 小结：社会性欲望的本质

让我们把这三种欲望放在一起看：

| 欲望 | 进化来源 | 满足时的爽感 |
|------|---------|------------|
| 被接纳 | 被接纳=存活，被驱逐=死亡 | 归属感 |
| 地位 | 地位高=资源多=存活繁衍优势 | 成就感、优越感 |
| 被需要 | 被需要=受保护=存活优势 | 价值感 |

**这些社会性欲望不是矫情，不是脆弱，不是过度在意别人。它们是进化刻在我们基因里的生存程序。**

在原始社会，这些欲望直接关系到你的生死。虽然在现代社会，被朋友孤立不会让你物理死亡，但你的大脑不知道——它还在用几万年前的程序运行。

这就是为什么被人批评时你会心跳加速，被人孤立时你会感到痛苦，被人认可时你会感到快乐。**你的生理反应是真实的，因为你的基因认为这事关生死。**

## 4.6 欲望的膨胀：从部落到现代社会

如果人类一直停留在原始部落阶段，事情可能还比较简单。部落就那么大，地位竞争的范围有限，被接纳的需求容易满足。

但人类社会继续演化了。

**农业革命**：人类定居下来，形成更大的村落和城镇。人口增加，社会分层出现，地位的阶梯变得更长、更复杂。

**文明的崛起**：国家形成，阶层固化，出现了贵族、平民、奴隶。地位欲望的对象从"部落里最强的猎人"变成了"社会阶层的上升"。

**工业革命**：财富成为地位的核心指标。金钱可以量化，可以比较，可以无限积累。欲望的天花板被打开了。

**信息时代**：社交媒体让比较的范围无限扩大。你不再只和身边的人比，而是和全世界的人比。你看到的都是别人精心筛选的高光时刻，你的参照系被严重扭曲。

**欲望随着社会复杂度的增加而膨胀。**

原始人想要吃饱，现代人想要米其林三星。
原始人想要被部落接纳，现代人想要百万粉丝。
原始人想要在部落里有地位，现代人想要财务自由、阶层跃升。

更关键的是，现代人的欲望几乎**全部指向他人**：

- **财富**：钱本身是一张纸，它的价值来自于别人愿意用东西和你交换。
- **地位**：地位是一种社会评价，没有他人的承认，地位不存在。
- **名声**：名声存在于别人的脑子里，不是你自己能制造的。
- **爱情**：爱情是两个人的游戏，单方面不成立。
- **被认可**：认可必须来自他人，你不能自己认可自己（那叫自恋）。

**当欲望全部指向他人时，一个残酷的事实就浮现了：你无法自产快乐。**

## 4.7 依赖注入：为什么我们无法自产快乐

现在让我们回到那个精神病人的例子。

他为什么能自产快乐？因为他的欲望系统和社会**脱钩**了。他不需要真实的财富，不需要真实的认可，不需要真实的爱情。他的大脑构建了一个虚拟世界，在那个世界里，他的欲望被满足了。

但我们做不到。

**因为我们的欲望系统已经被社会性重写了。**

我们需要真实的认可，而不是自己想象的认可。
我们需要真实的爱，而不是自己想象的爱。
我们需要真实的成就，而不是自己想象的成就。

用软件领域的语言说，我们的快乐模块有一个**外部依赖**（External Dependency）。它需要从外部系统（他人、社会）获取输入，才能正常运行。

```
// 精神病人的代码
function getHappiness() {
    return self.generateHappiness();  // 自产自销
}

// 正常人的代码  
function getHappiness() {
    let recognition = others.recognize(self);  // 依赖他人认可
    let love = others.love(self);              // 依赖他人的爱
    let status = society.evaluate(self);       // 依赖社会评价
    return process(recognition, love, status); // 处理后输出快乐
}
```

**每个正常人都是"舔狗"。**

我们需要从他人那里获取某些东西，才能满足我们的欲望，才能感到快乐。

你可以"舔"爱情——渴望被爱人认可和接纳。
你可以"舔"事业——渴望被资本和市场认可。
你可以"舔"社会——渴望被大众尊重和羡慕。

形式不同，本质一样：**快乐的原材料来自外部。**

这是人类作为社会动物的底层架构。你无法修改这个架构，就像你无法修改自己的基因。

但你可以选择：**你要向谁请求这些依赖？你要注入哪些外部模块？**

## 4.8 多态性：快乐的配置因人而异

虽然每个人都需要从外部获取快乐的原材料，但**配置比例因人而异**。

有些人，爱情占60%，事业占20%，友情占20%。对他们来说，爱情顺利，整个人生就亮了；爱情失败，其他再好也没用。

有些人，事业占60%，家庭占30%，其他占10%。对他们来说，事业成功是第一优先级，为此可以牺牲很多其他东西。

有些人，家庭占50%，稳定占30%，自由占20%。对他们来说，老婆孩子热炕头就是最大的幸福。

**没有哪种配置是"正确的"，也没有哪种配置是"错误的"。**

崇尚爱情和崇尚金钱，本质上没有高下之分。它们只是不同的欲望配置，对应不同的人生追求。

那些说"爱情至上"的人，没有资格嘲笑追求财富的人。
那些说"事业为重"的人，没有资格嘲笑享受生活的人。

**每个人的目标函数不一样，你没有权力用你的目标函数去评判别人的人生。**

这里的关键是：**你要认清自己的配置是什么。**

很多人痛苦的原因是，他们以为自己想要A，其实内心深处想要B。他们追求了很久的A，得到了，却发现不快乐。

- 他以为自己想要高薪，拼命工作，年入百万了，却发现自己真正想要的是自由时间。
- 她以为自己想要稳定，考了公务员，进了体制，却发现自己真正想要的是挑战和成长。
- 他以为自己不在乎爱情，一心搞事业，功成名就了，却发现自己真正想要的是有人懂他。

**这种错配是最大的悲剧：你中了彩票，但发现那不是你想要的奖品。**

所以，在追求之前，先搞清楚：你的欲望配置到底是什么？什么才能真正让你快乐？

这不是一个容易回答的问题。因为社会一直在告诉你"你应该想要什么"，而不是帮你发现"你真正想要什么"。

**还有一个更隐蔽的陷阱：某些看似纯粹的欲望，其实并不纯粹。**

我们常常歌颂"无私的爱"——父母对子女的爱，爱人之间的扶持，老师对学生的付出。

但如果我们诚实地拆解这些"无私"，会发现它们依然是欲望满足的过程：

- 父母为子女付出，看到子女快乐、成功，父母会"爽"。这种爽感来自于"我的付出产生了价值"。
- 爱人之间相互扶持，看到心爱的人幸福，自己会"爽"。这种爽感来自于"我让我爱的人更好了"。
- 老师倾注心血培养学生，看到学生取得成就，老师会"爽"。这种爽感来自于"我的教导开花结果了"。

**这些"无私"的付出，本质上依然是从外部获取快乐原料。** 只不过获取的方式是"通过让他人快乐来让自己快乐"。

这没有任何问题。这恰恰是人类社会能够运转的底层机制——我们的快乐和他人的快乐绑定在一起，所以我们愿意合作、愿意付出、愿意爱。

**但问题在于：很多人错估了自己的欲望纯度。**

他们以为自己是"纯粹的付出，不求回报"，但内心深处其实有一个隐藏的期望：我付出了，对方应该按我期望的方式回应。

- 父母为子女付出，期望子女听话、感恩、按自己规划的路走。当子女叛逆、不领情、走了另一条路，父母会感到愤怒和挫败："我白付出了。"
- 爱人之间付出，期望对方同等回报、同等在意。当对方没有给出预期的回应，就会感到不平衡："我对你这么好，你怎么能这样对我？"
- 老师付出精力，期望学生成才、感恩师恩。当学生平庸，老师会感到失落："我的心血白费了。"

**这就是"投资型欲望"的陷阱。**

你以为你在"无私地给予"，其实你在"投资并期待回报"。你的付出是本金，你期望的回应是利息。

但问题是：**这笔投资的回报率，不由你决定。**

- 子女是独立的个体，他们不欠你一个"按你期望活"的人生。
- 爱人有自己的感受方式，他们不一定能给出你想要的那种回应。
- 学生有自己的命运，他们的成就不完全取决于你的教导。

**外部因素是不可控的。当你把快乐押注在他人的回应上，你就把方向盘交给了别人。**

这不是说不要付出，不要爱。而是说：

**第一，诚实地认识自己的欲望。** 你的付出是纯粹的给予，还是期待回报的投资？不要自我欺骗。

**第二，如果是投资，就做好亏损的准备。** 投资有风险，不是每一笔都能回本。如果你接受不了"付出了却得不到期望的回应"，那就降低投入，或者调整期望。

**第三，区分"可控"和"不可控"。** 你能控制的是你的付出，不能控制的是对方的回应。把注意力放在可控的部分，对不可控的部分保持平常心。

这又回到了前面章节的所说的：**成功 = 策略 × 概率。** 你的付出是策略，对方的回应是概率。尽人事，听天命。

**而此策略是否合理完全取决于对自身欲望的认知程度，换句话说就是要正视自己的自私，不要自我感动。**

## 4.9 正视欲望：不要对你的源代码感到羞耻

说到这里，不得不讨论一个障碍：**社会对欲望的污名化。**

前文也已经论述了，所谓的“无私奉献”只是欲望寄托于他人之上的一种表现形式。

但是从小到大，我们接受的教育多数都是：欲望是不好的，要克制欲望，要无私奉献，要淡泊名利。

贪财？可耻。
好色？下流。
虚荣？肤浅。
想出人头地？功利。

这套规训让很多人对自己的欲望产生了**羞耻感**。他们明明想要钱，却不敢承认；明明想要被认可，却要装作不在乎；明明想要爱情，却要表现得很清高。

**这种压抑不会让欲望消失，只会让它变形。**

压抑对财富的欲望，可能变成对"清贫"的道德优越感（本质还是在追求某种优越感）。
压抑对认可的欲望，可能变成"我不需要任何人"的假性独立（本质是害怕得不到认可）。
压抑对爱情的欲望，可能变成工作狂（用一种欲望替代另一种欲望）。

被压抑的欲望不会消失，它会在暗处积累能量，然后以扭曲的方式爆发出来。

**欲望本身是中性的。它是进化写在你基因里的源代码，是驱动你行动的底层程序。**

你不需要为自己的源代码感到羞耻。

想要钱不可耻——钱代表资源和安全感，想要它是正常的。
想要被认可不可耻——被认可代表社会价值，想要它是正常的。
想要爱情不可耻——爱情代表亲密连接，想要它是正常的。
想要成功不可耻——成功代表能力证明，想要它是正常的。

**可耻的不是欲望本身，而是用不正当的手段去满足欲望。**

想要钱，通过努力工作赚取，正当。通过欺诈偷盗获取，不正当。
想要认可，通过做出成绩获得，正当。通过造假吹牛获得，不正当。
想要爱情，通过真诚付出赢得，正当。通过欺骗操控获得，不正当。

**正视你的欲望，理解你的欲望，然后用正当的方式去追求它。**

这是一个成年人应该有的态度。

## 4.10 沙盒机制：法律是游戏规则，不是道德枷锁

现在我们来讨论最后一个问题：**欲望的边界在哪里？**

如果每个人都无限制地追求自己的欲望，会发生什么？

答案是：**丛林社会，弱肉强食。**

你想要那块肉，我也想要，怎么办？打一架，谁赢谁拿。
你想要那个配偶，我也想要，怎么办？打一架，谁赢谁娶。
你想要那块地，我也想要，怎么办？打一架，谁赢谁占。

这是自然界的法则。狮子不会和羚羊讲道德，它饿了就吃。

但人类社会发展出了一套不同的系统：**法律和规则。**

法律的本质是什么？

**法律不是消除弱肉强食，而是改变弱肉强食的方式。**

在没有法律的丛林里，强弱由拳头决定。
在有法律的社会里，强弱由规则框架内的能力决定。

你不能用暴力抢别人的钱，但你可以通过商业竞争赚到比别人更多的钱。
你不能强迫别人和你结婚，但你可以通过提升自己吸引到更好的伴侣。
你不能用武力夺取地位，但你可以通过能力和努力获得更高的职位。

**法律没有让世界变得"公平"，它只是让竞争的规则变得明确。**

就像一个沙盒游戏，法律划定了边界：在边界内，你可以尽情玩耍；越过边界，系统会惩罚你。

这个边界是什么？简单说就是：**你追求自己的欲望时，不能侵犯他人的合法权益。**

你可以追求财富，但不能偷不能抢不能骗。
你可以追求爱情，但不能强迫不能骚扰不能破坏他人家庭。
你可以追求地位，但不能造谣不能陷害不能玩弄权术伤害他人。

在这个边界内，**你追求你的欲望，让自己活得更爽，是完全正当的。**

不需要羞耻，不需要内疚，不需要假装自己是个圣人。

## 4.11 小结：重新编译你的人生目标

让我们回顾这一章的核心逻辑：

**活着是被动的，追求"爽"是主动的。** 你没有选择出生，但你可以选择如何度过这段runtime。活着的唯一诚实目标，是让自己的主观体验更好——也就是"爽"。

**爽感来自欲望满足。** 欲望是因，爽感是果。没有欲望就没有爽感。

**欲望从动物性演化到社会性。** 人类最初的欲望是生存和繁衍，形成社会后，演化出了被接纳、地位、被需要等社会性欲望。

**现代人的欲望几乎全部指向他人。** 财富需要他人承认，地位需要他人评价，爱情需要他人回应。我们无法自产快乐，因为快乐的原材料必须从外部获取。

**这是进化的结果，不是缺陷。** 每个正常人都是"舔狗"，都需要从他人那里获取某些东西才能感到满足。这是人类作为社会动物的底层架构。

**但欲望的配置因人而异。** 有人爱情优先，有人事业优先，有人家庭优先。没有对错之分，只有是否清楚自己真正想要什么。

**不要对欲望感到羞耻。** 欲望是写在基因里的源代码，是驱动你行动的底层程序。压抑欲望不会让它消失，只会让它变形。正视它、理解它、用正当的方式追求它。

**法律是游戏规则，不是道德枷锁。** 在不侵犯他人权益的前提下，追求你的欲望、让自己活得更爽，是完全正当的人生。

**所以，你应该做什么？**

搞清楚你的欲望配置——什么真正能让你快乐？
接受你需要外部依赖的事实——然后选择依赖谁、依赖什么。
正视你的欲望——不要假装自己是个没有欲望的圣人。
在规则内全力追求——不羞耻，不内疚，不虚伪。

**诚实地活着，比什么都重要。**

我们人生中所做的一切选择，最终也都应该服务于让自己过得更“爽”。在达成此共识的基础上，我们对后续的人生问题的探讨才有意义。请正视自己的“自私”，并用正当的方式去追求它。我从不否认“无私奉献”，值得社会的宣传和赞扬。但是与此同时，我们也必须清醒地认识到，这些所谓的“无私”行为只是达成“自私”目的的过程中惠及他人罢了，要杜绝“刻意式无私奉献”的伪善和自我感动。
